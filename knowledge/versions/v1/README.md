# Archivo histórico

Estos documentos describen decisiones antiguas de V1. Se conservan para entender
la historia del producto, no como fuente normativa actual para nuevas tareas,
arquitectura o implementacion.

Conceptos historicos o retirados en esta seccion:

- `tenant`
- `branch`
- `handoff` como modelo principal
- `solution_template` como centro del producto

La fuente actual debe usar el modelo vigente:

- `organization`
- `account`
- `bot`
- interacciones
- actions
- guardrails
- colaboración con staff
- pendientes

Para trabajo actual, consultar primero `knowledge/AGENTS.md`,
`knowledge/IMPLEMENTATION_STATUS.md` y los documentos vigentes en
`knowledge/architecture/`.
